# MRes-Research-Project
This repository aims to share the information of the main results and the source codes of MRes research project: Influence of the Boreal Summer Intraseasonal Oscillation on Monsoon Rainfall and Soil Moisture over the Western Ghats, India. 
1. The source codes for data processing are saved in **Script** directory. The technical information about this project can be found through https://haichen6622.github.io/WesternGhats/.
2. The main research result is displayed with a poster.
