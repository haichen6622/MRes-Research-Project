# MRes-Research-Project
This repository aims to share the information of the main source codes and the final results of MRes research project: Influence of the Boreal Summer Intraseasonal Oscillation on Monsoon Rainfall and Soil Moisture over the Western Ghats, India. 
1. The key source codes for data processing are saved in **Script** directory. The technical information about this project can be found through https://haichen6622.github.io/WesternGhats/.
2. The final research result is displayed with a poster,https://doi.org/10.5281/zenodo.4302608.
